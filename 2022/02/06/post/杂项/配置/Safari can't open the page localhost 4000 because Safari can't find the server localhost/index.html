Safari can’t open the page localhost: 4000 because Safari can’t find the server localhost

几个月没碰 hexo 博客了，今天准备上来看看，结果发现 Safari 访问不了 localhost了，127.0.0.1 也无法访问。

看一眼 host 文件，可能是之前为了上 GitHub 调 host 把默认的覆盖了，百度了一份 mac 默认 host 文件加上就好了。
