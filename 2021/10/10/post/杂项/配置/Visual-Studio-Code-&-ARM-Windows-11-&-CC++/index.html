Visual Studio Code & ARM Windows 11 & C/C++

Created2021-10-10|Updated2022-02-06|杂项配置
|Post View:

mingw

这个网站下载 👉 https://bit.ly/mingw10

因为我对 Windows 的一些系统路径不是很熟悉，我就直接把 mingw 装在 C 盘根目录下了。

安装好之后，拷贝 mingw/bin 的路径，添加到环境变量。

选中 Path ，Edit。

New 之后拷贝进去保存即可。

Code Runner

和在 mac 配置一样，修改设置使其能一次性编译所有源文件。

Debug

和在 mac 配置一样。

看来以后 debug 要开 windows 虚拟机才行 🤡
