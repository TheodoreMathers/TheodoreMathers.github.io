Hexo 博客搭建

Created2021-08-02|Updated2022-02-06|杂项
|Post View:

本地部分

1.在官网下载安装 node.js

https://nodejs.org/en/

下载左边长期支持版。

安装好之后得到一个 node.js 本身，一个 npm 包管理器

安装好之后可以在终端运行 npm -v 检查一下.

Nom 在国内速度较差，安装 cnpm 使用国内源。

安装需要 sudo 权限

2.安装 hexo 框架

3.创建本地博客

在用户个人路径下创建博客路径，在路径下初始化 hexo（在新电脑上还原以前的博客时，先新建空的路径初始化好 hexo 之后，再把原来的文件复制进来）

4.初始化

远端部署

1.GitHub 新建repo

theodoremathers.github.io

一定要和自己用户名相同，一定要小写

2.安装插件

在本地路径进行安装

3.设置 _config.yml

默认的情况下每次 deploy 都要输入账号密码，GitHub 更新之后每次还要输入 token，很不方便，因此选择通过 ssh 进行连接。

在本地生成密钥之后，拷贝密钥

将密钥粘贴到 GitHub 的设置中

同时，修改 _config.yml 中的 repo 字段，改写成仓库的 ssh 链接地址

4.部署

主题

直接在 github 搜索 hexo theme 之类的东西就可以找到各种主题，大多数在首页都可以找到安装命令，安装好之后到 _config.yml 里面修改主题。

坑1⃣️

安装了一个主题之后访问白屏报错：

extends includes/layout.pug block content include ./includes/mixins/post-ui.pug #recent-posts.recent-posts +postUI include includes/pagination.pug

百度一下可能是缺依赖之类的，安装一下

可能会跳出来提示有漏洞要修复，直接无视。

坑2⃣️

用 npm 安装了一个主题之后 themes 文件夹里面居然是空的？？？

还是老老实实用 git clone 把文件都 down 过来。
