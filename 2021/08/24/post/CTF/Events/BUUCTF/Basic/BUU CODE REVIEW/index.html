分析

  1. md5 值用弱类型比较绕过
    1. PHP 当中使用 == 来进行比较的时候，系统会自动处理数据类型， 进行分析是数字比较还是字符比较。 而当一个字符串值是 0e 开头的时候，就会被当成数值。而 0exxxxxx 都为0
    2. 那么就只需要找到两个不同的数值，md5 值的开头是以 0e 开头的即可
